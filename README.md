## Development Setup Local

1. Clone this repository
```bash
git clone git@github.com:AlexGalhardo/boilerplate-react-vite-bootstrap5.git
```

2. Enter repository
```bash
cd boilerplate-react-vite-bootstrap5/
```

3. Run setup.sh
```bash
chmod +x setup.sh && ./setup.sh
```
